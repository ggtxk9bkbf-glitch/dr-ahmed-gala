# Dr. Ahmed Gala Clinic — Landing Page

Professional landing page for Dr. Ahmed Gala Clinic, an advanced aesthetic and skin-care clinic in Cairo. Built with React + Vite + Tailwind CSS.

## Stack

- React 19 + Vite
- Tailwind CSS 3 (Plus Jakarta Sans, custom palette)
- IntersectionObserver-driven scroll animations

## Sections

1. Sticky frosted-glass header with backdrop-blur
2. Hero — headline, doctor portrait, floating badges, stats row
3. About — doctor bio with credential grid
4. Popular Services — 8 image cards
5. Why Choose Us — 3-tab switcher with treatment cards
6. Excellence in Every Aspect — 4-column feature grid
7. Patient Testimonials — horizontal scroll on mobile, grid on desktop
8. Before & After Results — split comparison cards
9. Contact form + Get In Touch
10. Dark footer with brand, social, link columns

## Design tokens

| Token | Value |
| --- | --- |
| Font | Plus Jakarta Sans |
| Primary | `#2d5a4e` (deep green) |
| Accent | `#c9a87c` (gold) |
| Background | `#f9f7f4` (cream) |
| Text | `rgb(45 52 54)` |
| Header bg | `rgb(248 250 249 / 0.9)` + `backdrop-filter: blur(12px)` |

## Scripts

```bash
npm install    # install dependencies
npm run dev    # start Vite dev server
npm run build  # production build

```
